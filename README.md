> yet another fed tool

## Install

```bash
npm install -g dong
```

## Usage

### init

```bash
dong init

```

> initialize scaffolds for web, spm, seajs etc.

### serve

```bash
dong serve

```

> start a web server with static serve and api mocking.

### build

```bash
dong build

```

> build view and static files.
